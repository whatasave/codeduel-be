# Codeduel Backend

## Makefile commands 

### GO commands
```bash
make run

make build

make run

make dev

# run tests using GO test
make test
```

### Docker commands
```bash
# build docker image
make docker-build

# start docker container with env file and 5000 port mapping 
make docker-up

# stop docker container
make docker-down

# delete and recreate the docker container
make docker-restart
```
